# low-level-ml

Low-Level ML is an educational systems programming project focused on implementing numerical computing and machine learning algorithms from first principles.

The project starts with a C++23 linear algebra core, then expands into classical machine learning algorithms and a future Rust implementation for comparison.

## Goals

- Build matrix and vector operations from scratch
- Implement classical machine learning algorithms
- Explore performance optimization in C++
- Compare C++ and Rust implementations
- Practice real library design, testing, and benchmarking
